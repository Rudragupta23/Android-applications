# ▶️ How to Run Any Project

1. Clone the repository:

   ```
   git clone https://github.com/your-username/android-projects.git
   ```

2. Open Android Studio.

3. Click on **Open**.

4. Navigate into any project folder (example: `BeautifulSplashApp`).

5. Click **Run** to execute the application on an emulator or physical device.
